# Personal Knowledge Vault

A simple terminal-based note-taking app.

## What It Does

Store and manage your notes from the command line. Create, view, search, edit, and delete notes easily.

## How to Run

```bash
python knowledge_vault.py
```

## Features

- Create notes
- View all notes
- Search notes by keyword
- Edit existing notes
- Delete notes
- Auto-saves to JSON file

## Requirements

- Python 3

## Usage

Follow the menu prompts:
1. Create Note - Add a new note
2. View All Notes - See all your notes
3. Search Notes - Find notes by keyword
4. Edit Note - Update a note
5. Delete Note - Remove a note
6. Exit - Close the program

## Files

- `knowledge_vault.py` - Main program
- `notes.json` - Where notes are stored (created automatically)

## Example

```
Title: My First Note
Content: Hello World!
```

That's it! Simple and easy to use.
